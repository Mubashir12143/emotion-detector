# Emotion Detector

## Project Overview
Emotion Detector is a Python application that uses IBM Watson NLP to identify emotions from text input. It can analyze sentences or phrases and categorize them into emotions such as joy, sadness, anger, fear, and surprise.

## Features
- Detects multiple emotions from text input
- Provides formatted output for easy readability
- Includes error handling for invalid or blank input
- Deployable as a web application using Flask
